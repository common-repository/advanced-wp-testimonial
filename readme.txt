Easily Manage the Testimonials with WordPress Site 

== Description ==

with the Help of this Plugin user can easily Setup the Testimonial Section.this Section is Featured with
the WordPress "Custom Post Type" and For the sack of extensibility Plugin Uses the Major Default WordPress Functions and naming conversation 
So developers Can Easily Extend the Plugin as per their Requirement. 

= Features =
* Plugin Allow to customize Style of the Single Testimonial Page
* Plugin Allow to customize Style of the Testimonial Archive Page
* Plugin Allow to customize Style of the Testimonial Widget Page
* Change the URL for the Testimonial Part


== Installation ==
1. Upload "advanced-wp-testimonial" folder to the "/wp-content/plugins/" directory
2. Activate the plugin "Advanced WP Testimonial" through the 'Plugins' menu in WordPress
3. You can Customize the Settings Under the Testimonial -> Settings 

== Frequently asked questions ==

= Style =
Instead of modify the Core Template file copy the Template file from the Plugins Template Directory to Your current Active theme and modify
those files only.
